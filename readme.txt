Override official AMP WordPress plugin.

== Description ==

This plugin overrides some parts of WordPress AMP plugin to provide a customized user experience. Changes include: 

- Removing **powered by WordPress** from the footer.
- Removing **gravatar** image in author meta on single post page.
- Removing **google fonts**, reducing calls to them.
- Using **georgia** as body font.

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
